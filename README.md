Documentation
-------------

Wiki6 is WYSIWYG wiki.

### Prepare

1.	Change Password.  
	Default Password is **wiki6**, please change.  
	`echo hash("sha512","yourpassword")`  
	write in `etc/setting.json`  
	Will be fixed.

2.	Setting Permissions.  
	Need R/W permissions to `etc` and `var`  

3.	Enabling Rewrite Module.
